# General Descent Beyond ThetaZ

Status: Open Frontier.

Known closed result:

thetaZ (pachner23 T a b c p q) lam < thetaZ T lam
iff
vertexDeg T p + vertexDeg T q <= 10

Therefore universal descent for thetaZ cannot be asserted globally from this route.

Any stronger unconditional theorem must introduce a different invariant,
a restricted move domain, or an alternate monotonic quantity.
