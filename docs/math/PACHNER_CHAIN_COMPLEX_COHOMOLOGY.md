# Conditional: Pachner chain-complex cohomology

We model a sampled Pachner move groupoid as a chain complex:
- vertices = triangulations
- edges = Pachner moves

Boundary operator:
\[
\partial_1: C_1 \to C_0
\]

Cohomology:
\[
H^0 = \ker(\partial_0)/\operatorname{im}(\partial_1)
\]

In the sampled model, this computes only a local approximation to the global invariant space.
