# Nonlocal Support Theorem

Goal:
Prove that edges outside the Pachner 2→3 local region have zero incidence change.

Statement:
If e is not in the local edge set L, then
Δ_inc(e) = 0.

Role:
This isolates all nontrivial behavior to the finite local edge set,
enabling a complete finite classification of Δ_inc(e).
