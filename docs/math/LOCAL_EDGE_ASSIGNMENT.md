# Local Edge Assignment

Assignment (prototype):

Removed edges:
{a,b}, {b,c}, {c,a} → Δ = -1

Created edges:
{p,q} → Δ = +1

Incident edges:
{a,p}, {b,p}, {c,p}, {a,q}, {b,q}, {c,q} → Δ = +1

Nonlocal edges:
Δ = 0

Note:
This is a structural prototype assignment.
Exact combinatorial validation is still required.
