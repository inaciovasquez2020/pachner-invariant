# Edge Incidence Case Split

Goal:
Replace trivial delta with explicit local combinatorics.

Edge classes in a Pachner 2→3 move:

1. Edges removed: Δ = -1 or -2
2. Edges preserved: Δ = 0
3. Edges created: Δ = +1 or +2

Constraint:
Δ_inc(e) ∈ {-2,-1,0,1,2}

Next:
Assign each edge in the local configuration to one of these classes.
