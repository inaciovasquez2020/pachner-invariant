# Structural Boundary

Status: LOCKED

This repository formalizes a combinatorial invariant Θ(T, λ) on triangulated 3-manifolds and its behavior under Pachner moves.

It does not claim:
- Full PL classification of 3-manifolds
- General manifold recognition algorithms
- Geometric classification (e.g., hyperbolic structures)

All results are restricted to the combinatorial/triangulation regime.
