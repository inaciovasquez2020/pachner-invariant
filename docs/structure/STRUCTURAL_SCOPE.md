# Structural Scope

## Core Scope
- Finite triangulations of 3-manifolds
- Pachner 2→3 moves (Valid23)
- Edge-degree and vertex-degree bookkeeping
- Theta invariant: Θ(T, λ)
- Local-to-global descent mechanism

## Support Scope
- Piecewise-linear topology (triangulation language)
- Algorithmic 3-manifold topology

## Non-Core Scope
- Hyperbolic geometry (not required for invariant)
- Knot invariants
- Full algebraic topology machinery beyond basic invariants
